## Fanclub cards

React app that uses API and presents data in a responsive way.

Main features:

- Fetch users from API
- Present data as user cards
- Filter users' cards
- Add/Delete user
- Use tachyons library for styling components

[See live](https://miloszcwen.github.io/fans/)
